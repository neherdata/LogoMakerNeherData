# Free Online LogoMaker

It is a desktop app version of Free Online Logo maker https://logomakr.com/

## Setup

```bash
sudo npm install -g electron 
```
## Clone the Repo

```bash
git clone https://github.com/JeFcorp/LogoMaker.git
```

## Install Dependencies

```bash
cd LogoMaker
npm install
```

## Run App
```bash
cd LogoMaker
./logo-maker
```
